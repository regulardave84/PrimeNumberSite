How to Run
-------------------------------

1.) Download and install the latest version of Ruby
2.) Once Ruby is installed open your command prompt and enter 'gem install cucumber'
3.) In the command prompt enter 'gem install rspec'
4.) In the command prompt enter 'gem install capybara'
5.) Download and install Firefox v33 (Capybara will NOT work for the latest version of Firefox)
6.) In Firefox turn auto updates off.
7.) Download the Prime Number Site repository to your machine
8.) In the command prompt navigate to the PrimeNumberSite folder and enter cucumber features/prime_number_url.feature
9.) The feature should run using Firefox and all tests should pass
